HTML

HTML stand for Hypertext Markup Language, it is the language used to create websites in this column I will go over some terms commonly used in HTML.

Tag

Tags are used to hold content, they can hold all types of content from text to images.

Class

Tags may have one or more classes, seperated by spaces. Classes can be selected in css to style the tag they are in.

Id

Tags can have an Id to identify it, like Classes, Id's can be selected in css to edit them, however unlike classes a tag should never have more than one id.

Href

The Href attribute holds URL's and tells the browser where to go, these are used in navigation bars, but are also used in linking the css to the html.

Nesting

Nesting is when a tag is place within a tag, this is used a lot with navigation bars with li's (list tag) being placed within ul's (unordered list tag).

CSS

CSS stands for Cascading Style Sheets, in this column I will go over some terms commonly used in CSS.

Selector

Selectors are used in CSS to select the part of the html that is being styled, it can be a tag, class name, id, pseudo class, etc.

Property

Properties are defined within selectors by defining a property and a value, these are seperated by a ":" and are ended with a ";".

Property Value

Property Value is the value given to the property to style the element.

Pseudo Classes

Pseudo Classes allow for tags to be styled in certain cercumstances for example ":hover" this pseudo class styles the tag when the mouse is hovered over it.

Child Selectors

Child Selectors allow for the selection of certain elements vis parental nesting, for exaple using the ">" .
